Privacy Policy

Hakase Anime Publisher — Last updated: June 2026

1. Overview

Hakase Anime Publisher ("the App") is a personal, non-commercial automation tool. This policy explains how the App handles data in the context of TikTok API integration.

2. Data We Collect

The App does not collect any personal data from users. The only data handled is:

  • An OAuth access token issued by TikTok, stored locally on the author's machine in a file named .tiktok_credentials.json.
  • Video files created by the author, uploaded directly to their own TikTok account via the Content Posting API.

3. Data Sharing

No data is shared with any third party. OAuth tokens are never transmitted to any server other than TikTok's own API endpoints.

4. Data Storage

All data (credentials, videos) is stored exclusively on the author's local machine. No cloud storage or external databases are used by the App itself.

5. TikTok API Compliance

The App uses the TikTok Content Posting API in accordance with TikTok's Terms of Service and TikTok API Developer Policies.

6. Contact

For questions, contact the author via the GitHub repository: github.com/m-matheus/multi-agent-video-publisher
